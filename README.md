X-Ray-AI is an AI-powered web application that analyzes chest X-ray images and predicts whether signs of pneumonia are present. Using a deep learning model trained on medical imaging data, the system can provide fast screening results with approximately 88% accuracy.

What It Does
Upload a chest X-ray image
AI analyzes the image in seconds
Returns a pneumonia prediction and confidence score
Designed as an educational demonstration of AI in healthcare
Tech Stack
Python
TensorFlow
Replit
HTML/CSS/JavaScript
